# **ScannerTest**
## Description
在編譯器中，token 是組成程式的最小單位，需要由 Scanner 先把 input text 按照規
則轉為 Token，才能讓 Parser 進行處理。請依照題目規則，試著寫一個 Scanner 來
取得 Tokens 並印出。
  
| Terminal  | Regular Expression | 
|-------|:-----:|
| NUM | ( [1-9][0-9]* ) \| 0 |
| PLUS  | \\+ |
| MINUS | \\- |
| MUL | \\* |
| DIV | \\\\ |
| LPR | \\( |
| RPR | \\) |
